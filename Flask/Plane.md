# CapriceTravel 旅行网站后台系统规划

## 一、系统概述

CapriceTravel 是一个旅行规划网站后台系统，用于管理用户的旅行计划、文件夹分类及旅行路线信息。

---

## 二、功能模块

### 2.1 用户认证模块

| 功能 | 描述 |
|------|------|
| 用户注册 | 支持邮箱/手机号注册，密码加密存储 |
| 用户登录 | 支持邮箱/手机号登录，JWT Token 认证 |
| 密码找回 | 通过邮箱验证码重置密码 |
| 用户信息管理 | 修改昵称、头像、个人简介等 |

---

### 2.2 旅行文件夹模块

| 功能 | 描述 |
|------|------|
| 创建文件夹 | 用户可创建旅行文件夹进行分类管理 |
| 编辑文件夹 | 修改文件夹名称、描述、封面 |
| 删除文件夹 | 删除空文件夹或连同内部旅行计划一起删除 |
| 文件夹列表 | 查看用户所有文件夹（支持分页、排序） |
| 文件夹详情 | 查看文件夹内的所有旅行计划 |

---

### 2.3 旅行计划模块

#### 2.3.1 旅行基本信息

| 字段 | 类型 | 描述 |
|------|------|------|
| id | UUID | 旅行计划唯一标识 |
| name | String | 旅行名称 |
| folder_id | UUID | 所属文件夹ID（可为空，表示未分类） |
| user_id | UUID | 所属用户ID |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

#### 2.3.2 旅行路线信息

| 字段 | 类型      | 描述 |
|------|---------|------|
| start_point | String | 起始地点（地址） |
| end_point | String  | 结束地点（地址） |
| waypoints | Array   | 中转点列表 |
| total_distance | Float   | 总距离（公里） |
| estimated_duration | Integer | 预计总时长（分钟） |

#### 2.3.3 途径点详情

每个途径点包含以下信息：

| 字段 | 类型 | 描述             |
|------|------|----------------|
| id | UUID | 途径点唯一标识        |
| name | String | 地点名称           |
| location | Object | 位置信息（经纬度、详细地址） |
| type | String | 类型（起点/中转/终点）   |
| reserved_time | Integer | 预留时间（1-24小时）   |
| arrival_time | DateTime | 预计到达时间         |
| departure_time | DateTime | 预计出发时间         |
| notes | String | 备注             |
| order | Integer | 排序序号           |


---

### 2.4 时间预留功能

#### 预留时间规则
- 最小预留时间：1小时
- 最大预留时间：24小时
- 默认预留时间：2小时
- 用户可在1-24小时范围内自由调整每个途径点的预留时间

#### 时间计算逻辑
```
总时长 = 行驶时间 + Σ(每个途径点的预留时间)
```
